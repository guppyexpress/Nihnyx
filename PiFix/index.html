PiFix, A cure to the beatsaber issues found on Pimax, Xtal, Starvr and Valve Index devices while Parallel projections is disabled. When this mod is installed to your beatsaber client; the shader that causes double vision is shifted to the side. This shift is in an exact specific value so that the double vision problem is solved.
The lighting properly lines up on each eye as well as the reflections. This eliminates the double vision appearence and greatly reduces the pain on the user.

The only current bug is the reflections are not centered. This is a side effect of having to shift the lighting to the left/right by a very specific value, the reflections will be fixed in a later update.
In order to install this BeatSaber mod, it requires BSIPA to be installed, or have downloaded and used Mod Assistant for BeatSaber.
